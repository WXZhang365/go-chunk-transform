### 使用方法:
gctu -h

```
Go语言开发的文件传输和验证工具,支持文件切分,文件合并,文件校验等功能

Usage:
  gctu [command]

Available Commands:
  check       校验文件完整性
  help        Help about any command
  md5         计算单个文件的hash值
  merge       合并文件
  split       用于将文件切分为对应的块大小

Flags:
  -h, --help   help for gctu

Use "gctu [command] --help" for more information about a command.
```
